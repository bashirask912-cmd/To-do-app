✅ TaskFlow – Modern To-Do App

A sleek, responsive, and feature-rich To-Do List Web App built with HTML, CSS, and Vanilla JavaScript. TaskFlow helps you manage daily tasks efficiently with priority levels, drag-and-drop organization, and dark mode support.

📸 Preview

✨ Features
📝 Add, edit, and delete tasks
✅ Mark tasks as completed
🎯 Priority levels (Low, Medium, High)
📊 Real-time task statistics
🌙 Dark / Light mode toggle
💾 Local storage (data persists after refresh)
🔄 Drag & drop task reordering
🧹 Clear completed tasks
⚡ Smooth animations & modern UI
📱 Fully responsive design
🔌 Basic PWA support (Service Worker)
🛠️ Tech Stack
HTML5
CSS3 (Custom properties, Flexbox, Grid)
JavaScript (ES6+)
Font Awesome (icons)
Google Fonts (Inter)

🚀 Getting Started
1. Clone the repository
git clone  https://github.com/bashirask912-cmd/To-do-app/new/main?filename=README.md
2. Open the app
cd taskflow
open index.html

Or simply double-click the index.html file.

🎮 How to Use
✍️ Enter a task in the input field
🎯 Select priority (Low / Medium / High)
➕ Click Add Task
✅ Click circle to mark complete
✏️ Edit or 🗑️ delete tasks
🔄 Drag tasks to reorder
🌙 Toggle dark mode
🧹 Use Clear Completed button to clean up
📊 Task Statistics

TaskFlow automatically tracks:

Total tasks
Completed tasks
Pending tasks
High-priority tasks
